# Ravi Medicose – Digital Pharmacy System

A lightweight, single-page digital pharmacy platform built for a local medical store in Panchkula, India.  
It transforms a traditional WhatsApp-based ordering system into a structured online catalog with cart management and automated order tracking.

---
DEPLOYED- https://papa-enj1mz7ph-asmis-projects-63413446.vercel.app/

## Problem Statement

Local pharmacies often rely on WhatsApp messages and phone calls for taking orders. This creates several issues:

- No centralized order history
- Difficulty tracking repeat customers
- Manual errors in medicine quantity and pricing
- No structured system for business insights or reporting

This results in an unorganized workflow and poor scalability for the business.

---

## Solution

This project digitizes the pharmacy workflow by providing a simple web-based ordering system.

Customers can:
- Browse available medicines in a structured catalog
- Search and filter products by category
- Add items to a cart and manage quantities
- Place orders via WhatsApp or checkout form

On the business side:
- All orders are automatically recorded in Google Sheets using an API integration
- This allows the shop owner to maintain a proper digital record of all transactions

---

## Features

- Interactive shopping cart system
- Real-time search and category filtering
- Medicine catalog with prescription (Rx) tagging
- Automatic discount logic for selected items
- WhatsApp click-to-chat order integration
- Checkout modal with order summary
- Google Sheets integration for order tracking
- Embedded Google Maps for store location
- Mobile-responsive design for real-world usage

---

## Tech Stack

- HTML5
